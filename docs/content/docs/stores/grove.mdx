title: Grove Store
description: SQL store using grove ORM.
---

The Grove store uses the [Grove](https://github.com/xraph/forgery/grove) ORM for database access. It provides struct-based models and supports PostgreSQL.

## Usage

```go
import "github.com/xraph/relay/store/grovestore"

s, err := grovestore.New(grovestore.Config{
    DSN: "postgres://user:pass@localhost:5432/relay?sslmode=disable",
})
if err != nil {
    log.Fatal(err)
}

if err := s.Migrate(ctx); err != nil {
    log.Fatal(err)
}

r, err := relay.New(relay.WithStore(s))
```

## Internals

| Aspect | Detail |
|--------|--------|
| ORM | Grove ORM with struct-based models |
| Migrations | Grove migrator with Go-defined migrations |
| Transactions | Database-level ACID |
| Ping | `db.PingContext(ctx)` |
| Close | Closes the connection pool |

## When to use

- When you prefer an ORM over raw SQL.
- Teams using grove in their Forge stack.
